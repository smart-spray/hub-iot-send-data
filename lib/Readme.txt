Após instalar a biblioteca AzureIoT, substituir os arquivos : Esp32MQTTClient.cpp e Esp32MQTTClient.h pelos arquivos de mesmo nome que estão nessa pasta.
O path para encontrar os arquivos da biblioteca AzureIoT, podem mudar dependendo de como foi instalado a IDE do Arduino:

C:\Users\{UsuáriodoSistema}\Documents\ArduinoData\packages\esp32\hardware\esp32\1.0.6\libraries\AzureIoT\src       
Ou 
C:\Users\{UsuáriodoSistema}\AppData\Local\Arduino15\packages\esp32\hardware\esp32\1.0.6\libraries\AzureIoT\src
